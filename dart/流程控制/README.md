# 流程控制
## 条件语句
```dart
if (condition) {
  // do something
} else if (condition) {
  // do something
} else {
  // do something
}
```
## 循环语句
### for 循环
```dart
for (var i = 0; i < 10; i++) {
  // do something
}
```
### while 循环
```dart
while (condition) {
  // do something
}
```
### do...while 循环
```dart
do {
  // do something
} while (condition);
```
## 跳转语句
### break 语句
```dart
for (var i = 0; i < 10; i++) {
  if (i == 5) {
    break;
  }
  // do something
}
```
### continue 语句
```dart
for (var i = 0; i < 10; i++) {
  if (i == 5) {
    continue;
  }
  // do something
}
```
